# S3 MCP (S3-compatible: AWS S3 / Cloudflare R2 / MinIO)

Auth: access key + secret.

Write actions are **write-gated** (must pass `confirm:true`).

## Env
- `S3_ACCESS_KEY_ID`
- `S3_SECRET_ACCESS_KEY`
- `S3_REGION` (default `us-east-1`)
- `S3_ENDPOINT` (optional; set for R2/MinIO)
- `S3_BUCKET` (default bucket for tools)
- `S3_FORCE_PATH_STYLE` (optional; useful for some S3-compatible endpoints)

## Tools
- `s3_list_buckets`
- `s3_list_objects`
- `s3_get_object_text`
- `s3_put_object_text` (**confirm:true**)
- `s3_presign_get`

## Run
```bash
cd servers/s3-mcp
npm i
export S3_ACCESS_KEY_ID="..."
export S3_SECRET_ACCESS_KEY="..."
export S3_REGION="us-east-1"
# export S3_ENDPOINT="https://<accountid>.r2.cloudflarestorage.com"  # for R2
export S3_BUCKET="your-bucket"
node src/index.js
```
